# 🩺 Sehhaty 2.0 — صحتي في كل مكان

مقترح تطوير لمنصة **صحتي** السعودية لنقلها من تطبيق يُفتَح عند الحاجة إلى منظومة صحية محيطية ذكية تتكامل مع حياة المواطن اليومية — داخل المنزل، على الطاولة، وفي السيارة.

**[🔗 افتح العرض المباشر (GitHub Pages)](#)**

> حدّث الرابط أعلاه بعنوان Pages بعد النشر، مثال:
> `https://USERNAME.github.io/sehhaty-2/`

---

## 🎯 العروض التوضيحية

| الديمو | التقنية | الوصف |
|---|---|---|
| [🪞 المرآة الذكية](demo-mirror.html) | MediaPipe FaceLandmarker + rPPG | قياس النبض والعلامات الحيوية من لون الجبهة بدون تلامس |
| [🏋️ مدرب التمارين](demo-trainer.html) | MediaPipe PoseLandmarker | عدّ التكرارات من زوايا المفاصل الحقيقية (سكوات، بوش أب، بلانك) |
| [🥗 الشيف الصحي](demo-chef.html) | AI + سلة تسوق حية | خطة وجبات مخصصة وفق الملف الصحي + ربط مع البقالة |
| [🚗 داش كام السيارة](demo-dashcam.html) | FaceLandmarker + EAR | كشف نعاس حقيقي من إغماض العين + بروتوكول ٩٣٧ |
| [🔧 تشخيص الكاميرا](diagnostic.html) | — | أداة لفحص سبب عدم عمل الكاميرا |

---

## 🧪 التقنيات المستخدمة

- **MediaPipe Tasks Vision** — نماذج FaceLandmarker و PoseLandmarker على الحافة
- **Canvas API** — استخراج إشارة rPPG من البكسلات
- **getUserMedia** — الوصول المباشر للكاميرا
- **Vanilla JS + HTML/CSS** — بدون إطارات خارجية
- **IBM Plex Sans Arabic** — الخط الرسمي

---

## 📋 المتطلبات

- متصفح حديث: Chrome 90+ / Safari 16+ / Edge / Firefox 92+
- كاميرا وإضاءة جيدة
- اتصال إنترنت (لتحميل نماذج MediaPipe أول مرة)
- **HTTPS أو localhost** — لن تعمل الكاميرا عبر `file://`

---

## 🏠 التشغيل محليًا

```bash
python3 serve.py
```
ثم افتح المتصفح على: `http://localhost:8765/index.html`

بديل على Mac: اضغط مرتين على `تشغيل-الديمو.command`
بديل على Windows: اضغط مرتين على `تشغيل-الديمو.bat`

## ☁️ النشر على GitHub Pages

راجع الدليل المفصّل: [**نشر-على-github-pages.md**](نشر-على-github-pages.md)

باختصار:
1. أنشئ repo جديد على GitHub
2. ارفع ملفات المشروع
3. من Settings → Pages → اختر `main` branch
4. خلال ~٢ دقيقة يصبح متاحًا على `https://USERNAME.github.io/REPO-NAME/`
